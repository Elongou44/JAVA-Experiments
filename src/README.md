# Java学习项目

## 项目简介

本项目是一个Java学习代码库，包含了从基础语法到高级特性的各种Java编程示例。这些代码是在大学一年级Java课程中完成的作业，旨在帮助学习者理解Java的核心概念和编程技巧。

## 目录结构

项目按照知识点进行了分类，主要包括以下目录：

- **basic-syntax**: 基础语法和控制结构
- **arrays-collections**: 数组和集合
- **oop**: 面向对象编程
- **io**: IO操作
- **multithreading**: 多线程
- **gui**: GUI编程
- **others**: 其他杂项

## 包含的知识点

### 基础语法和控制结构 (basic-syntax)
- 变量和数据类型
- 运算符
- 控制语句（if-else, switch, for, while）
- 方法定义和调用
- 字符串操作
- 日期时间处理
- 包装类
- 简单算法实现

### 数组和集合 (arrays-collections)
- 数组的定义和操作
- 数组排序
- ArrayList的使用
- HashSet的使用
- Map的使用
- 对象数组

### 面向对象编程 (oop)
- 类和对象
- 构造方法
- 封装
- 继承
- 多态
- 抽象类
- 接口
- 异常处理
- 自定义异常

### IO操作 (io)
- 文件路径操作
- 字符流和字节流
- 对象序列化
- 文件读写
- 编码转换

### 多线程 (multithreading)
- 线程的创建和启动
- 线程同步
- 线程生命周期

### GUI编程 (gui)
- JFrame窗口创建
- 菜单设计
- 事件处理

## 如何运行项目

1. 确保您已经安装了Java开发工具包（JDK）
2. 使用命令行或IDE（如Eclipse、IntelliJ IDEA）打开项目
3. 编译并运行相应的Java文件

## 总结

本项目涵盖了Java编程的各个方面，从基础语法到高级特性。通过学习这些代码示例，您可以更好地理解Java的核心概念和编程技巧，为后续的Java开发打下坚实的基础。

希望这些代码对您的Java学习有所帮助！
