# Java 作业集合

这是我在大学一年级学习Java课程时完成的作业集合，包含了从基础语法到面向对象编程、多线程、IO操作等多个方面的练习。

## 项目结构

```
src/
├── arrays_collections/    # 数组和集合相关练习
├── basic_syntax/         # 基础语法练习
├── gui/                  # 图形界面相关练习
├── io/                   # 输入输出操作练习
├── multithreading/       # 多线程练习
├── oop/                  # 面向对象编程练习
└── others/               # 其他练习
```

## 目录说明

### arrays_collections
- 包含数组操作、集合框架使用等练习
- 如ArrayList、HashSet、Map等集合的使用
- 数组排序等算法实现

### basic_syntax
- 基础语法练习，包括：
  - 计算器实现
  - 日期差值计算
  - 斐波那契数列
  - 杨辉三角
  - 各种基础算法练习

### gui
- 图形界面相关练习
- 包含JFrame和菜单相关实现

### io
- 输入输出操作练习
- 文件读写
- 对象序列化
- 编码转换

### multithreading
- 多线程编程练习
- 线程创建和管理

### oop
- 面向对象编程练习
- 继承、多态、抽象类、接口
- 异常处理
- 自定义异常

### others
- 其他练习
- 如倒计时器等功能实现

## 技术栈

- Java SE
- 集合框架
- 多线程
- IO操作
- 图形界面（Swing）

## 如何使用

1. 克隆仓库到本地
2. 使用IDE（如IntelliJ IDEA、Eclipse等）打开项目
3. 运行各个目录下的示例代码
